life-ledger 软件架构图

单二进制 Go Web 服务，React 静态资源内嵌，SQLite 作为唯一数据源，Caddy 负责 HTTPS 反向代理。

SAD: docs/dev/sad.md Diagram: docs/dev/sad-diagram.html Status: draft / 2026-07-04

访问与部署

  • • Caddy 负责 HTTPS、域名和反向代理。
  • • 应用默认只监听 127.0.0.1。
  • • React dist 通过 Go embed 打进二进制。

应用边界

  • • API 层负责请求响应，不直接写 SQL。
  • • 领域服务负责校验、事务和业务规则。
  • • Excel 导入先整表校验，再事务写入。

数据与安全

  • • SQLite 是唯一运行时数据源。
  • • Cookie 登录态配合 CSRF 保护写操作。
  • • 审计事件写入 audit_events，日志只做脱敏摘要。
